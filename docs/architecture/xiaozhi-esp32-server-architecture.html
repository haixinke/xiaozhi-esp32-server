小智ESP32服务器系统架构

AI虚拟宠物 · 情感陪伴 · 智能语音助手

核心服务

  • xiaozhi-server - Python AI 核心 (端口 8000/8003)
  • Provider 模式管理 AI 流水线
  • 支持 ASR·LLM·TTS·VAD·Intent·Memory
  • WebSocket 实时语音对话
  • 三层配置机制 (本地·文件·远程)

管理后台

  • manager-api - Java Spring Boot REST (端口 8002)
  • 设备注册·配置管理·用户认证
  • MyBatis-Plus + MySQL + Redis
  • Apache Shiro 双认证机制
  • Liquibase 数据库迁移

数据存储

  • MySQL - 业务数据 (设备·用户·配置)
  • Redis - 缓存·会话·系统参数
  • OceanBase - PowerMem 记忆存储
  • 向量存储 + 知识图谱
  • 用户画像与情感陪伴数据

前端应用

  • 智控台 - Vue.js 2 Web 管理控制台 (端口 8001)
  • 移动端 - Uni-app 跨平台 (H5/小程序/iOS/Android)
  • REST API 统一接口 (/xiaozhi)
  • OAuth2 Bearer Token 认证

外部集成

  • ASR - 阿里云·百度·腾讯·讯飞
  • LLM - OpenAI·通义千问·Gemini·Ollama
  • TTS - 阿里云·火山引擎·Fish Speech
  • RAGFlow - 知识库适配器
  • 多供应商灵活切换

安全特性

  • OAuth2 - Web/移动端用户认证
  • 服务密钥 - xiaozhi-server 机器对机器认证
  • XSS / SQL 注入防护
  • HTTPS 加密传输
  • 集中式错误码管理
