孵化互动 & API 集成

用户点击加速孵化，破壳时调用后端

孵化互动设计
孵化进度机制
孵化进度 0%
点击规则：
  • 每次点击蛋增加 3~5% 进度（随机，增加趣味）
  • 同时也有 自动增长（约 0.5%/秒），不点也能孵完
  • 用户点击产生 波纹 + 音效反馈
蛋的反馈：
  • 0~30%：轻微摇晃，偶尔晃动
  • 30~60%：摇晃加剧，出现细小裂纹
  • 60~90%：剧烈摇晃，裂纹增多变粗
  • 90~100%：蛋壳发光，即将破壳
点击视觉反馈：
  • 蛋被点击时 弹性缩放（scale 1.0 → 1.08 → 1.0）
  • 点击位置出现 爱心粒子 向上飘散
  • 进度条旁显示 "+N%" 飘字
出生 API 集成
破壳时调用流程
// 破壳动画播放到一半时触发
POST /xiaozhi/pet/birth
// Request Body:
{ "deviceId": "当前设备ID" }
// Response:
{
  "nickname": "小团子",
  "birthDate": "2026-05-08 ...",
  "bazi": { "year":"丙午", ... },
  "mbti": "INFP",
  "zodiac": "taurus"
}
API 调用时机：
  • 进度 100% → 破壳动画开始
  • 动画播放到 50% 时发起 API 请求
  • API 返回后，兔子弹出 + 宠物信息卡出现
出生展示卡片：
  • 宠物昵称（大字）
  • MBTI 性格类型
  • 星座
  • 出生时间
  • 展示 3~4 秒后自动淡出，进入聊天界面
错误处理：
  • API 失败：显示 Toast 提示，仍展示兔子但用默认数据
  • 设备未注册：提示"请先配置设备信息"
